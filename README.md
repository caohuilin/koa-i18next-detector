<p align="center"><big>
A i18next language detecting plugin for Koa 2.0 framework.
</big></p>


## Install

```sh
npm i -D koa-i18next-detector
```

## Usage

```js
import koaI18nextDetector from "koa-i18next-detector"

koaI18nextDetector() // true
```
